# Governix Overview

Governix is the tenant-aware control, metering, and audit layer for Bedrock-powered AI.

Governix is not a Bedrock proxy. An integrating application still calls Bedrock directly.

Governix adds three control-plane capabilities:

1. tenant-aware request policy evaluation
2. per-tenant usage, quota, and showback
3. audit-grade retrieval and generation ledger events
